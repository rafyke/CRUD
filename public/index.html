Meu Primeiro CRUD - AMIGO OCULTO

Projeto desenvolvido como prática didática em HTML,CSS e Javascript, utilizando os frameworks Bootstrap pela praticidade na interface responsiva e EMAILJS para envio de e-mails diretamente do cliente.

Um CRUD (Create, Read, Update, Delete) é uma abreviação que representa as operações básicas de um sistema de gerenciamento de banco de dados. Ele é importante porque permite a criação, leitura, atualização e exclusão de dados, formando a base para a funcionalidade completa de muitos aplicativos e sistemas. O CRUD é essencial para a interação e manipulação eficiente dos dados.
